# @ktmcp/bclaws

MCP server for BC Laws

## Installation

```bash
npm install -g @ktmcp/bclaws
```

## Usage

Add to your MCP settings:

```json
{
  "mcpServers": {
    "bclaws": {
      "command": "mcp-bclaws"
    }
  }
}
```
